## Installation

### Prerequistes


### 1. Clone

```shell
$ git clone https://github.com/minji-dev/SCG-2022-hackathon.git
```

### 2. Install Packages

```bash
$ npm install
```
```
$ npm i -g @nestjs/cli
```
```
$ npm i -g @nestjs/mapped-types
```
### 3-1. Start development server

```shell
$ npm start
```

서버 시작 후, 
[`http://localhost:3001`](http://localhost:3001) 에 접속
