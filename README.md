## Installation

### Prerequistes


### 1. Clone

```shell
$ git clone https://github.com/minji-dev/SCG-2022-hackathon.git
```

### 2. Install Packages

```bash
$ npm install
```
### 3-1. Start development server

front / back 각각 서버 실행

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```


서버 시작 후, 
[`http://localhost:3000`](http://localhost:3000) 에 접속
